# BookSearch

한 줄 소개: 네이버 책 검색 API를 활용한 책 검새 애플리케이션
팀 구성: 안드로이드 앱 개발 1명

## 🔗 공부한 내용 정리
[블로그 링크](https://blog.naver.com/pck4949/223382513406)

## 🖥 Development

### Features
- 책 검색
- 책 정보 표시
- 책 정보 클릭 시 웹페이지로 연결 (개발 예정)

## 📖 상세 내용
Jetpack Compose와 MVI 패턴 적용을 위한 간단한 책 검색 애플리케이션입니다.

## 🛠️ 사용 기술 및 라이브러리
- Kotlin
- **Coroutines**
- **orbit-mvi(MVI 패턴)**
- Retrofit2
- OkHttp3
- ViewModel, LiveData
- Glide
- **Jetpack Compose**
- **Hilt - 의존성 주입**

### Architecture
orbit 라이브러리를 활용하여 MVI 패턴을 적용하였습니다.
